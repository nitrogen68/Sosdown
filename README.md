# ⬇️ Social Downloader

All-in-One Social Media Downloader — Educational & Non-Commercial.

## Supported Platforms
- Instagram (Reels, Posts, Stories)
- TikTok (Videos — No Watermark)
- YouTube (Shorts, Videos)
- Twitter / X
- Facebook
- Threads

## Features
- ✅ Auto-detect platform from URL
- ✅ Preview: thumbnail, title, duration, author
- ✅ Multiple formats: MP4 HD/SD, MP3, Image HD
- ✅ No Watermark for TikTok & Instagram Reels
- ✅ Download history (LocalStorage)
- ✅ Dark / Light mode
- ✅ Mobile-first responsive design
- ✅ Free API integration

## Free APIs You Can Use
1. **RapidAPI** — [Social Media Download](https://rapidapi.com/) (100 req/day free)
2. **SnapSave** — Free tier available
3. **SSSTik** — TikTok only, free
4. **Y2mate** — YouTube only, free

## Quick Start

```bash
# 1. Clone
git clone https://github.com/yourusername/social-media-downloader.git
cd social-media-downloader

# 2. Install
npm install

# 3. Setup env
cp .env.example .env
# Edit .env with your API keys

# 4. Run
npm run dev
